---TITLE---
BMI Calculator

---DESCRIPTION---
Write a function called `calculateBMI` that takes two parameters: `weight` (in kilograms) and `height` (in meters). It should return the Body Mass Index (BMI), rounded to **1 decimal place** as a number.

The formula is: `BMI = weight / (height * height)`

Example:
- Input: `calculateBMI(70, 1.75)`
- Output: `22.9`

This question reviews **functions**, **arithmetic**, and **number formatting**.

---INITIALCODE---
function calculateBMI(weight, height) {
  // Your code here
}

---TESTCASES---
[
  { "func": "calculateBMI(70, 1.75)", "expected": 22.9, "type": "return" },
  { "func": "calculateBMI(60, 1.6)", "expected": 23.4, "type": "return" },
  { "func": "calculateBMI(90, 1.8)", "expected": 27.8, "type": "return" }
]

---HINTS---
- Calculate the BMI using the formula `weight / (height * height)`.
- Use `.toFixed(1)` to format to 1 decimal, but remember it returns a string. Wrap with `Number(...)` or `parseFloat(...)` to return a number.

---EXPLANATION---
`.toFixed(n)` rounds a number to `n` decimal places but returns a string. We convert it back to a number so the test cases pass strict equality with numeric expected values.

```javascript
function calculateBMI(weight, height) {
  const bmi = weight / (height * height);
  return Number(bmi.toFixed(1));
}
```

---SOLUTION---
function calculateBMI(weight, height) {
  const bmi = weight / (height * height);
  return Number(bmi.toFixed(1));
}
